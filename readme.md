# Install localy

1. Download the archive or clone the repository
2. Unpack the archive
3. Run the following command:

```
yarn install
```
4. Run the following command:

```
sequelize db:create
```
5. Run the following command:

```
sequelize db:migrate
```
6. Run the following command:

```
yarn start
```
